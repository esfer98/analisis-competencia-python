# Análisis de Competencia Local (Python)

Este proyecto busca evaluar locales competidores cercanos, analizando su ubicación, horas pico de demanda y el sentimiento de las reseñas de los clientes.

## Objetivos del Proyecto
* **Geolocalización:** Identificar competidores directos en un radio específico.
* **Análisis de Tráfico:** Evaluar horas pico para entender el flujo de clientes.
* **Reputación Online:** Analizar reseñas y puntuaciones para detectar fortalezas y debilidades.

## Estructura del Proyecto
* `src/`: Lógica principal (extracción y procesamiento).
* `data/`: Almacenamiento local de archivos procesados (CSV/JSON).
* `tests/`: Pruebas de código.
* `notebooks/`: Experimentación y visualización de datos.

## Instalación
1. Clonar el repositorio.
2. Crear un entorno virtual: `python -m venv venv`.
3. Instalar dependencias: `pip install -r requirements.txt`.
